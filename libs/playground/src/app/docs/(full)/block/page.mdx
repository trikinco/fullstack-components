## Blocks

The UI above is made out of the following blocks:

```jsx
<Block
  prompt={`A navbar with a modern SVG logo, ${NAME_LIB} and some links. Dark bg, light text.`}
  fallback={
    <Spinner
      className="flex gap-3 p-3 items-center"
      classNameSpinner="mb-0"
    >
      Creating navbar...
    </Spinner>
  }
/>
```

```jsx
<Block
  prompt={`A sleek UI hero section with some text about ${NAME_LONG}. Dark bg, light text.`}
  fallback={
```

## Setup

1. Add the `handleBlockRequest` to the [API handler route](/docs/get-started/#setup).

```ts title="app/api/fsc/route.ts"
import {
  handleFSComponents,
  handleBlockRequest,
  type FSCOptions,
} from '@trikinco/fullstack-components'

const fscOptions: FSCOptions = {
  handleBlock: handleBlockRequest({
    openAiApiKey: process.env.OPENAI_API_KEY || '',
  }),
  // Additional options and handlers...
}

const fscHandler = handleFSComponents(fscOptions)

export { fscHandler as GET, fscHandler as POST }
```

2. Import and use the `Block` component from `/client`.

```jsx
import { Block } from '@trikinco/fullstack-components/client'
```
